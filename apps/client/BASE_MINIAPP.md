# Base Mini App - Environment Variables

Set these in `apps/client/.env.local` (client-safe) for previews and manifest:

- `NEXT_PUBLIC_URL` (ex: https://your-domain.example)
- `NEXT_PUBLIC_ACCOUNT_ASSOC_HEADER`
- `NEXT_PUBLIC_ACCOUNT_ASSOC_PAYLOAD`
- `NEXT_PUBLIC_ACCOUNT_ASSOC_SIGNATURE`
- `NEXT_PUBLIC_BASE_OWNER_ADDRESS` (Base account address)
- `NEXT_PUBLIC_MINIAPP_NAME` (optional, default: DeFi Dungeon)
- `NEXT_PUBLIC_MINIAPP_WEBHOOK_URL` (optional)

Generate account association at the Base Build preview tool: https://www.base.dev/preview?tab=account
